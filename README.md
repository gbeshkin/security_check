# AI Security Scanner v3

Это версия с веб-интерфейсом, где можно:
- вставить ссылку на публичный GitHub-репозиторий
- проверить локальный проект по пути
- открыть историю последних отчётов
- скачать JSON / HTML / SARIF

## Что умеет

- принимает GitHub URL вида `https://github.com/owner/repo`
- клонирует репозиторий во временную папку
- запускает `scanner.py`
- сохраняет отдельный отчёт для каждого скана
- показывает результаты в браузере

## Ограничения

Через UI сейчас поддерживаются **публичные GitHub-репозитории**.

Для private repo потребуется отдельная доработка:
- GitHub OAuth
- или PAT/token
- или серверная интеграция через GitHub App

## Установка

### 1. Установить Python-зависимости
```bash
pip install -r requirements.txt
```

### 2. Установить внешние инструменты сканирования

#### Semgrep
```bash
pip install semgrep
```

#### Trivy
Установи по инструкции под свою ОС.

#### OSV-Scanner
Установи бинарник `osv-scanner` и добавь его в `PATH`.

Если какой-то инструмент не установлен, приложение всё равно запустится, но соответствующая часть анализа будет пропущена.

## Запуск

```bash
python web/app.py
```

Открой:
```bash
http://localhost:5000
```

## Поток работы

1. Открываешь главную страницу
2. Вставляешь GitHub URL
3. Нажимаешь кнопку сканирования
4. После завершения открывается страница отчёта
5. Можно скачать JSON / HTML / SARIF

## Что хранится

Каждый скан сохраняется в отдельную папку внутри `reports/<scan_id>/`:
- `security-report.json`
- `security-report.html`
- `security-report.sarif`
- `meta.json`
